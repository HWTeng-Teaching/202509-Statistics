# Z-Score Analysis for Exercise 2

## Problem Statement
For the data set with n=11 measurements: 2.3, 1.0, 2.1, 6.5, 2.8, 7.8, 1.7, 2.9, 4.4, 5.1, 2.0

Find:
- Mean
- Standard deviation
- Z-scores corresponding to minimum and maximum values
- Determine if there are possible outliers

## Solution

### Given Data
```
Data: [2.3, 1.0, 2.1, 6.5, 2.8, 7.8, 1.7, 2.9, 4.4, 5.1, 2.0]
n = 11
```

### Calculations

#### 1. Mean (μ)
```
μ = Σx / n = (2.3 + 1.0 + 2.1 + 6.5 + 2.8 + 7.8 + 1.7 + 2.9 + 4.4 + 5.1 + 2.0) / 11
μ = 38.6 / 11 = 3.51
```

#### 2. Standard Deviation (σ)
First, calculate the variance:
```
σ² = Σ(x - μ)² / n
```

Deviations squared:
- (2.3 - 3.51)² = 1.4641
- (1.0 - 3.51)² = 6.3001
- (2.1 - 3.51)² = 1.9881
- (6.5 - 3.51)² = 8.9401
- (2.8 - 3.51)² = 0.5041
- (7.8 - 3.51)² = 18.4041
- (1.7 - 3.51)² = 3.2761
- (2.9 - 3.51)² = 0.3721
- (4.4 - 3.51)² = 0.7921
- (5.1 - 3.51)² = 2.5281
- (2.0 - 3.51)² = 2.2801

```
σ² = 46.849 / 11 = 4.259
σ = √4.259 = 2.064
```

#### 3. Z-Scores for Minimum and Maximum Values

**Minimum value: 1.0**
```
z_min = (1.0 - 3.51) / 2.064 = -1.22
```

**Maximum value: 7.8**
```
z_max = (7.8 - 3.51) / 2.064 = 2.08
```

### Results Summary

|-----------|-------|

| Minimum Value | 1.0 |
| Maximum Value | 7.8 |

| Z-score (max) | 2.0792 |

### Outlier Analysis

**Criteria for outliers:**
- Moderate outliers: |z-score| > 2
- Extreme outliers: |z-score| > 3

**Analysis:**
- The minimum value (1.0) has a z-score of -1.22, which is within normal range
- The maximum value (7.8) has a z-score of 2.08, which slightly exceeds the moderate outlier threshold of 2
- **Conclusion:** The value 7.8 could be considered a mild potential outlier, but it's not an extreme outlier

## Z-分數分析 - 第2題

### 問題描述
對於練習1-3中的數據集，計算平均數、標準差，以及數據集中最小值和最大值對應的z-分數。這些z-分數是否表明這些數據集中存在可能的離群值？

### 第2題數據
**n = 11 個測量值：2.3, 1.0, 2.1, 6.5, 2.8, 7.8, 1.7, 2.9, 4.4, 5.1, 2.0**


### 計算過程

#### 1. 平均數 (μ)
```
μ = Σx / n = (2.3 + 1.0 + 2.1 + 6.5 + 2.8 + 7.8 + 1.7 + 2.9 + 4.4 + 5.1 + 2.0) / 11
μ = 38.6 / 11 = 3.5091
```

#### 2. 標準差 (σ)
首先計算變異數：
```
σ² = Σ(x - μ)² / n
```

各數值的偏差平方：
- (2.3 - 3.5091)² = 1.4619
- (1.0 - 3.5091)² = 6.2956
- (2.1 - 3.5091)² = 1.9856
- (6.5 - 3.5091)² = 8.9442
- (2.8 - 3.5091)² = 0.5028
- (7.8 - 3.5091)² = 18.4182
- (1.7 - 3.5091)² = 3.2730
- (2.9 - 3.5091)² = 0.3711
- (4.4 - 3.5091)² = 0.7936
- (5.1 - 3.5091)² = 2.5313
- (2.0 - 3.5091)² = 2.2768

```
σ² = 46.8541 / 11 = 4.2590
σ = √4.2590 = 2.0637
```

#### 3. 最小值和最大值的Z-分數

**最小值：1.0**
```
z_min = (1.0 - 3.5091) / 2.0637 = -1.2158
```

**最大值：7.8**
```
z_max = (7.8 - 3.5091) / 2.0637 = 2.0792
```

### 結果摘要

### 離群值分析

**離群值判斷標準：**
- 中等離群值：|z-分數| > 2
- 極端離群值：|z-分數| > 3

**分析結果：**
- 最小值 (1.0) 的z-分數為 -1.22，在正常範圍內
- 最大值 (7.8) 的z-分數為 2.08，略微超過中等離群值門檻2
- **結論：** 數值7.8可視為潛在的輕微離群值，但不是極端離群值

**完整Z-分數分析：**

### 離群值判斷結論

**是否存在可能的離群值：是的**

1. **潛在離群值**：數值7.8 (z-分數 = 2.08)
   - 略微超過2的門檻，可視為潛在離群值
   - 不是極端離群值（因為 |z-分數| < 3）

2. **數據分佈特徵**：
   - 大部分數據點（10個中的10個）在正常範圍內
   - 只有1個數據點為潛在離群值
   - 數據整體分佈合理，7.8可能代表一個較極端但不異常的測量值

### 統計意義

**Z-分數的意義：**
- z-分數表示數據點距離平均數多少個標準差
- |z-分數| < 1：約68%的數據應在此範圍內
- |z-分數| < 2：約95%的數據應在此範圍內  
- |z-分數| < 3：約99.7%的數據應在此範圍內

**本數據集的分佈：**
- 90.9% (10/11) 的數據在2個標準差內
- 100% (11/11) 的數據在3個標準差內
- 符合常態分佈的預期模式

**結論：**
Z-分數分析顯示數值7.8為潛在離群值，但不是極端離群值。這個數據集整體分佈合理，7.8雖然較為極端，但仍在可接受的統計範圍內。
