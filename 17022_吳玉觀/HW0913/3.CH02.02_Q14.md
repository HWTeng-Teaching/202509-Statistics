# 海水污染細菌計數分析

## 問題描述
海洋石油污染刺激某些類型細菌的生長。對10份海水樣本中的細菌計數（每100毫升）進行了測量。

### 分析任務：
a. 計算範圍
b. 計算 x̄ 和 s
c. 範圍約等於多少個標準差？

## 數據集
**細菌計數（每100毫升）：**
```
49, 70, 54, 67, 59, 40, 61, 69, 71, 52
```

## 解答

### a. 範圍計算

**排序後數據：**
```
40, 49, 52, 54, 59, 61, 67, 69, 70, 71
```

**範圍計算：**
- 最小值：40
- 最大值：71
- **範圍 = 71 - 40 = 31**

### b. 平均數和標準差計算

#### 平均數 (x̄)：
```
x̄ = (49 + 70 + 54 + 67 + 59 + 40 + 61 + 69 + 71 + 52) / 10
x̄ = 592 / 10 = 59.2
```

#### 標準差 (s) 計算步驟：

| 數值 (x) | x - x̄ | (x - x̄)² |
|----------|--------|-----------|
| 49 | -10.2 | 104.04 |
| 70 | 10.8 | 116.64 |
| 54 | -5.2 | 27.04 |
| 67 | 7.8 | 60.84 |
| 59 | -0.2 | 0.04 |
| 40 | -19.2 | 368.64 |
| 61 | 1.8 | 3.24 |
| 69 | 9.8 | 96.04 |
| 71 | 11.8 | 139.24 |
| 52 | -7.2 | 51.84 |
| **總和** | **0** | **967.60** |

**標準差計算：**
```
s² = Σ(x - x̄)² / (n - 1) = 967.60 / 9 = 107.51
s = √107.51 = 10.37
```

### c. 範圍與標準差關係

**計算範圍是多少個標準差：**
```
範圍 / 標準差 = 31 / 10.37 = 2.99 ≈ 3.0
```

**結論：範圍約等於 3 個標準差**

## 統計解釋

### 理論背景
根據統計學理論，對於常態分佈：
- 約68%的數據在 ±1個標準差內
- 約95%的數據在 ±2個標準差內  
- 約99.7%的數據在 ±3個標準差內

### 實際觀察
本數據集的範圍恰好約為3個標準差，這表明：

1. **數據分佈合理**：範圍在2-4個標準差之間是常見的
2. **無極端離群值**：所有數據點都在合理範圍內
3. **接近常態分佈**：符合統計理論預期

#### 答案總結：
a. 範圍 = 31  
- 最小值：40  
- 最大值：71  
- 範圍：71 - 40 = 31  
  
b. 平均數和標準差  
- 平均數 (x̄) = 59.2  
- 標準差 (s) = 10.37  

c. 範圍約等於 3 個標準差  
- 範圍/標準差 = 31/10.37 = 2.99 ≈ 3.0  
  
這個結果符合統計理論預期，表明數據分佈合理，無異常極端值。
