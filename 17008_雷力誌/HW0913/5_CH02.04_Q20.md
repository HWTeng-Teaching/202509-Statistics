最小值 (Minimum)

Min = X₍₁₎

最小值 = 1.70 微克/克

第一四分位數 (Q₁)

Q₁位置 = (n+1)/4

Q₁位置 = (28+1)/4 = 29/4 = 7.25

由於位置不是整數，需要在第7個和第8個數值間進行內插

第7個數值 = 118.0，第8個數值 = 168.0

小數部分 = 0.25

Q₁ = X₍₇₎ + 0.25 × (X₍₈₎ - X₍₇₎)

```
Q₁ = 118.0 + 0.25 × (168.0 - 118.0)
Q₁ = 118.0 + 0.25 × 50.0
Q₁ = 118.0 + 12.5
Q₁ = 130.5 微克/克
```

中位數 (Median, Q₂)

中位數位置 = (n+1)/2

中位數位置 = (28+1)/2 = 29/2 = 14.5

需要在第14個和第15個數值間進行內插

第14個數值 = 241.0，第15個數值 = 252.0

小數部分 = 0.5

中位數 = X₍₁₄₎ + 0.5 × (X₍₁₅₎ - X₍₁₄₎)

```
中位數 = 241.0 + 0.5 × (252.0 - 241.0)
中位數 = 241.0 + 0.5 × 11.0
中位數 = 241.0 + 5.5
中位數 = 246.5 微克/克
```

第三四分位數 (Q₃)

Q₃位置 = 3(n+1)/4

Q₃位置 = 3×(28+1)/4 = 3×29/4 = 87/4 = 21.75

需要在第21個和第22個數值間進行內插

第21個數值 = 316.0，第22個數值 = 318.0

小數部分 = 0.75

Q₃ = X₍₂₁₎ + 0.75 × (X₍₂₂₎ - X₍₂₁₎)

```
Q₃ = 316.0 + 0.75 × (318.0 - 316.0)
Q₃ = 316.0 + 0.75 × 2.0
Q₃ = 316.0 + 1.5
Q₃ = 317.5 微克/克
```

最大值 (Maximum)

最大值 = 485.0 微克/克

c. Are there any outliers?

1.5×IQR規則，該數據集中沒有檢測到統計學異常值

（Q₁=130.5, Q₂=246.5, Q₃=317.5），計算得出四分位距IQR = Q₃ - Q₁ = 317.5 - 130.5 = 187.0 μg/g

根據1.5×IQR規則，異常值的判定邊界為：

下界 = Q₁ - 1.5×IQR = 130.5 - 280.5 = -150.0 μg/g

上界 = Q₃ + 1.5×IQR = 317.5 + 280.5 = 598.0 μg/g

資料範圍：最小 1.70、最大 485.0，皆落在 '[−150.0, 598.0]' 之內，因此無離群值
