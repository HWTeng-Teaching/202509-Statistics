其中 1.70 μg/g 顯著低於其他值。
若知道前四隻海豚年齡 < 3 歲，而其他皆 > 8 歲，這可能反映：
年幼海豚尚未累積汞 → 生理代謝與暴露時間短
年長海豚因食物鏈累積汞 → 濃度較高
✅ 結論：年齡資訊提供了生物學上的合理解釋，支持這些低值並非測量錯誤，而是年齡差異造成的自然變異。
