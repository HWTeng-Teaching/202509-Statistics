# 1. Measures of Center (中心趨勢量數計算)

## 原始數據與排序
* **數據點個數 (n):** 5
* **原始數據:** 0, 5, 1, 1, 3
* **排序後數據:** 0, 1, 1, 3, 5

---

## 計算結果

### 1. 平均數 (Mean, $\bar{x}$)

**公式:** $$\bar{x} = \frac{\sum x}{n}$$

$$\sum x = 0 + 1 + 1 + 3 + 5 = 10$$

$$\bar{x} = \frac{10}{5} = 2$$

**平均數 (Mean): 2**

### 2. 中位數 (Median, M)

由於 $n=5$ (奇數)，中位數是第 $(5+1)/2 = 3$ 個數。

* 第 3 個數 (排序後): **1**

**中位數 (Median): 1**

### 3. 眾數 (Mode)

**定義:** 出現頻率最高的數值。數值 1 出現了兩次，其他數值只出現一次。

**眾數 (Mode): 1**
